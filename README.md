# Kimetsu Career Sprint RPG

Веб-приложение-игра для обучения, вдохновлённое "Истребителем демонов" (Kimetsu no Yaiba), разработанное для помощи в освоении фронтенд-разработки через геймифицированную ежедневную практику.

## Возможности

- **Ежедневные планы тренировок**: Автоматически генерируемые ежедневные задачи на основе вашего текущего цикла обучения
- **Прогрессия рангов**: Поднимайтесь от Мидзуното до Хаширы через постоянную практику
- **Циклы обучения**: 12 циклов, охватывающих JavaScript, TypeScript, React, Vue и многое другое
- **Практика интервью**: Режим спарринга для практики и пробные интервью с таймерами
- **Система серий**: Поддерживайте ежедневную последовательность для бонусного XP
- **Финальная битва**: Разблокируйте финальный вызов после освоения ключевых циклов

## Технологический стек

- **Next.js 14+** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Prisma** + **SQLite** (локальная база данных)
- **shadcn/ui** компоненты
- **framer-motion** (анимации)
- **lucide-react** (иконки)

## Начало работы

### Требования

- Node.js 18+ 
- npm или yarn

### Установка

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd DemonSlayer
```

2. Установите зависимости:
```bash
npm install
```

3. Настройте базу данных:
```bash
# Сгенерируйте Prisma client
npm run db:generate

# Запустите миграции
npm run db:migrate

# Заполните начальными данными
npm run db:seed
```

4. Запустите сервер разработки:
```bash
npm run dev
```

5. Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## Конфигурация

Настройки игры можно изменить в `config/game.ts`:

- `defaultTargetXp`: Ежедневная цель XP (по умолчанию: 100)
- `rankXpStep`: XP, необходимое за ранг (по умолчанию: 500)
- `streakBonusPer3Days`: Множитель бонуса за 3-дневную серию (по умолчанию: 0.05)
- `maxStreakBonus`: Максимальный бонус серии (по умолчанию: 0.25)
- `cycleAdvancementRequirements`: Требования для перехода к следующему циклу
- `mockTimerMinutes`: Таймер пробного интервью (по умолчанию: 20)

## Управление контентом

Контент хранится в JSON файлах в директории `/content`:

- **cycles.json**: Определения циклов и темы
- **questions.json**: Вопросы для интервью, организованные по циклам
- **taskTemplates.json**: Шаблоны задач для ежедневных планов

Вы можете редактировать эти файлы напрямую, чтобы настроить свой контент обучения без изменения кода.

## База данных

Приложение использует SQLite (локальная файловая база данных) с Prisma ORM. Файл базы данных находится в `prisma/dev.db`.

### Полезные команды

```bash
# Просмотр базы данных в Prisma Studio
npm run db:studio

# Сброс базы данных (ВНИМАНИЕ: удаляет все данные)
npx prisma migrate reset

# Создать новую миграцию
npm run db:migrate
```

## Разработка

### Добавление нового контента

1. **Вопросы**: Добавьте в `content/questions.json`
2. **Шаблоны задач**: Добавьте в `content/taskTemplates.json`
3. **Циклы**: Измените `content/cycles.json`

### Настройка UI

Приложение использует Tailwind CSS с кастомной темой Demon Slayer. Ключевые классы темы:

- `.katana-edge`: Подсветка верхней границы карточки
- `.breath-button`: Специальный стиль кнопки
- `.breathing-gauge`: Стиль прогресс-бара
